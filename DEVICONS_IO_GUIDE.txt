🔗 HOW TO GET ICONS FROM DEVICONS.IO:
================================================================================

METHOD 1: Using the API URL Pattern
------------------------------------
Pattern: https://devicons.io/api/[TECH-NAME]/[VERSION]

Examples:
Python:      https://devicons.io/api/python/original
React:       https://devicons.io/api/react/original
MySQL:       https://devicons.io/api/mysql/original
JavaScript:  https://devicons.io/api/javascript/original
Node.js:     https://devicons.io/api/nodejs/original

Versions available:
- original (colored)
- plain (monochrome)
- line (outline)

🔄 COMPARISON: DEVICON.DEV vs DEVICONS.IO
================================================================================

DEVICON.DEV (cdn.jsdelivr.net):
✓ More stable CDN
✓ Longer URLs
✓ Well-established
Example: https://cdn.jsdelivr.net/gh/devicons/devicon/icons/python/python-original.svg

DEVICONS.IO:
✓ Shorter URLs
✓ Cleaner API
✓ Same icons, different hosting
Example: https://devicons.io/api/python/original

DIRECT SVG:
✓ No external dependencies
✓ Always works (no CDN downtime)
✓ Larger database storage
Example: <svg xmlns="http://www.w3.org/2000/svg">...</svg>
