🚀 HOW TO USE ICONS:
================================================================================

STEP 1: Import the icon you want
---------------------------------
import { FaHome, FaUser, FaEnvelope } from 'react-icons/fa';

STEP 2: Use it as a React component
------------------------------------
<FaHome />
<FaUser size={24} />
<FaEnvelope className="text-blue-500" />

================================================================================
💡 COMMON USAGE PATTERNS:
================================================================================

Basic Icon:
-----------
<FaHome />

Custom Size:
------------
<FaHome size={24} />
<FaHome size={32} />

Custom Color:
-------------
<FaHome color="blue" />
<FaHome color="#3B82F6" />

With Tailwind Classes:
----------------------
<FaHome className="w-6 h-6 text-blue-500" />
<FaHome className="w-8 h-8 text-red-500 hover:text-red-700" />

In Buttons:
-----------
<button className="flex items-center gap-2">
  <FaHome />
  <span>Home</span>
</button>

With Hover Effects:
-------------------
<FaHome className="hover:text-blue-500 transition-colors" />
<FaHome className="hover:scale-110 transition-transform" />

Animated:
---------
<FaCog className="animate-spin" />
<FaHeart className="animate-pulse" />

🔍 HOW TO FIND ICONS:
================================================================================

METHOD 1: Search Online
------------------------
Visit: https://react-icons.github.io/react-icons/search
Search for any icon name (e.g., "home", "user", "mail")
Click on the icon to see its import statement

METHOD 2: Use the Icon Browser Component
-----------------------------------------
1. Import the IconBrowser component:
   import IconBrowser from './components/IconBrowser';

2. Add it to your app temporarily:
   <IconBrowser />

3. Browse visually and click to copy import statements

METHOD 3: Check the Reference File
-----------------------------------
Open: frontend/src/utils/iconReference.js
Contains categorized lists of common icons

METHOD 4: View Examples
------------------------
Open: frontend/src/examples/IconUsageExamples.jsx
See 10+ different usage patterns with code
