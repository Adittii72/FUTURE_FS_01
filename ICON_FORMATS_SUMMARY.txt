================================================================================
                    ICON FORMATS - QUICK REFERENCE
================================================================================

✅ YOUR PORTFOLIO NOW SUPPORTS 3 ICON FORMATS!

================================================================================
FORMAT 1: DEVICON.DEV (CDN URLs)
================================================================================

Pattern:
https://cdn.jsdelivr.net/gh/devicons/devicon/icons/[TECH]/[TECH]-[VERSION].svg

Examples:
Python:    https://cdn.jsdelivr.net/gh/devicons/devicon/icons/python/python-original.svg
React:     https://cdn.jsdelivr.net/gh/devicons/devicon/icons/react/react-original.svg
MySQL:     https://cdn.jsdelivr.net/gh/devicons/devicon/icons/mysql/mysql-original.svg

How to get:
1. Go to https://devicon.dev/
2. Search for technology
3. Click icon
4. Look for <img> tag section
5. Copy the URL from src attribute

================================================================================
FORMAT 2: DEVICONS.IO (Shorter API URLs) ⭐ NEW!
================================================================================

Pattern:
https://devicons.io/api/[TECH]/[VERSION]

Examples:
Python:    https://devicons.io/api/python/original
React:     https://devicons.io/api/react/original
MySQL:     https://devicons.io/api/mysql/original

How to get:
1. Go to https://devicons.io/
2. Search for technology
3. Note the technology name
4. Use pattern: https://devicons.io/api/[tech-name]/original

✓ Shorter URLs
✓ Cleaner format
✓ Same icons as devicon.dev

================================================================================
FORMAT 3: DIRECT SVG CODE ⭐ NEW!
================================================================================

Example:
<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 128 128">
  <path fill="#00758F" d="M2.001 90.458h4.108V74.235..."/>
</svg>

How to get:
1. Go to https://devicons.io/ or https://devicon.dev/
2. Search for technology
3. Click icon
4. Click "COPY SVG" button
5. Paste the ENTIRE SVG code in the icon field

✓ No external dependencies
✓ Always works (no CDN downtime)
✓ Paste directly from devicons.io

================================================================================
WHICH FORMAT SHOULD YOU USE?
================================================================================

Use DEVICONS.IO URLs if:
✓ You want shorter, cleaner URLs
✓ You're browsing devicons.io
✓ You want the simplest format

Use DEVICON.DEV URLs if:
✓ You prefer the established CDN
✓ You're already using devicon.dev
✓ You want maximum stability

Use DIRECT SVG CODE if:
✓ You want zero external dependencies
✓ You're concerned about CDN uptime
✓ You want to customize the SVG

ALL THREE WORK PERFECTLY! Choose what's easiest for you.

================================================================================

DIRECT SVG:
Code: <svg xmlns="http://www.w3.org/2000/svg">...</svg>
Length: Varies (500-2000 characters)

================================================================================
HOW TO ADD AN ICON (ANY FORMAT)
================================================================================

1. Login to your dashboard
2. Go to Skills section
3. Click "Add Skill" or "Edit"
4. Fill in:
   - Skill Name: Python
   - Category: Languages
   - Icon URL or SVG Code: [paste any of the 3 formats]
   - Proficiency: 85

5. Click "Save"

The system automatically detects which format you used! ✅

================================================================================
TROUBLESHOOTING
================================================================================

Icon not showing?
✓ Check if URL is accessible (paste in browser)
✓ Make sure SVG code is complete (starts with <svg and ends with </svg>)
✓ Try a different format
✓ Check browser console for errors (F12)

Which format is best?
✓ All work equally well
✓ Use whichever is easiest to get
✓ You can mix formats (some skills with URLs, some with SVG)

Can I change format later?
✓ Yes! Edit the skill and paste a different format
✓ The system automatically adapts
